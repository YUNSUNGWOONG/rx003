ROS2를 사용한 자율주행 기초
****************************************************************

이 문서는 제3회 국제대학생 EV 자율주행 경진대회에 참가하기위해, ROS2와 터틀봇3 버거를 사용한 자율주행을 구현하는 과정에서 조사한 정보의 기록이다.

문서의 구조는 유사한 과정을 기록한 논문인, "ROS를 활용한 서빙 이동로봇의 구현" :cite:`EUISEONKIM2019ROS` 참조하여 구성하였다.
